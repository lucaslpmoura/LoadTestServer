Servidor de teste de carga.

Rotas disponíveis:

  • /5kB - retorna uma página HTML de 5 kB de tamanho.
  • /50kB - retorna uma página HTML de 50 kB de tamanho.
  • /500kB - retorna uma página HTML de 500 kB de tamanho.
  • /stats - estatísticas do teste.
  • /stats - limpa o contador de requisiões usado no log.
